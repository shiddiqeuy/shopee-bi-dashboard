# AGENTS.md - Commit Message Rules & Guidelines

Kamu sedang bekerja pada issue GitHub/GitLab nomor #ISSUE_NUMBER.

**PERATURAN COMMIT MESSAGE (WAJIB):**

1. Setiap commit message harus mengandung `#ISSUE_NUMBER`
2. Letakkan reference di paling depan
3. Gunakan format konvensional:
   - `Fix #ISSUE_NUMBER: ...`
   - `Close #ISSUE_NUMBER: ...`
   - `Refactor #ISSUE_NUMBER: ...`
   - `#ISSUE_NUMBER: ...`

4. Deskripsi harus singkat, jelas, dan dalam bahasa Indonesia atau Inggris sesuai issue.

Contoh yang benar:
- `Fix #187: perbaiki bug login dengan Google`
- `#245: tambahkan pagination di daftar produk`

Contoh yang salah (jangan lakukan):
- `perbaiki bug login`
- `update fitur`

Sekarang kamu sedang garap issue #ISSUE_NUMBER. Selalu patuhi aturan ini.
